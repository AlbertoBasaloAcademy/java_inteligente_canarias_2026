## Programación Inteligente:

- **Evolución continua**: El software no se termina, se abandona o se evoluciona. 
- **Roles con Agentes**: Asignar responsabilidades específicas a la IA.
  - **Architect**: Visión global y orquestación.
  - **Builder**: Ejecución iterativa guiada por specs.
  - **Craftsman**: Mejora continua y refactoring.
